# Annual Appendix Sample Data

This folder contains the public-safe canonical annual inputs for the advanced appendix.

## Files

- `annual_appendix_inputs.csv`
  One row per project year with the annual price deck, throughput base, grade, recovery, unit costs, and capex fields used by the appendix.
- `appendix_parameters.csv`
  Scalar parameter table for commercial assumptions, fiscal proxy settings, ramp settings, and simulation settings.
- `appendix_scenarios.csv`
  Deterministic scenario registry used by the appendix page.
- `appendix_benchmark_metrics.csv`
  Canonical benchmark reference table used for benchmark transparency without requiring the workbook.

## Interpretation

- These files are sample/demo contracts for the appendix layer.
- They support planning-style valuation and downside discussion.
- They do not claim mine-plan realism, full fiscal realism, or asset-grade investment modeling.
