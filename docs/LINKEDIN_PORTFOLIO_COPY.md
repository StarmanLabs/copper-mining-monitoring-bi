## Short post

One of the projects I recently upgraded started as an Excel-based copper mining valuation model.

Instead of leaving it as a static workbook, I rebuilt the logic in Python and turned it into a reproducible analytical product with:

- deterministic valuation
- Monte Carlo simulation
- scenario analysis
- tornado sensitivity
- BI-ready fact and dimension tables

The most valuable part of the exercise was not just "coding the spreadsheet in Python". It was redesigning the project so it could be audited, extended, and used as the data layer for Power BI or Tableau dashboards.

That shift took the project from a useful workbook to a stronger portfolio-grade analytics asset.
